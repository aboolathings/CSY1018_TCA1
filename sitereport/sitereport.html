This assignment was my first experience with using html and css (outside of adding fancy things to my tumblr blog years ago!) and it was a lot of fun. I'm a very visual learner and creative individual, so being able to see my progress live with every line of code was both satisfying and rewarding. I much preferred this to learning Java for the first time, and no doubt I will most likely prefer this to javascript too! Now with no further ado... let's crack on with the report!

Firstly, I would like to detail my design choices and the websites I used to help with this process! I started off my picking the colour palette which I did through the help of coolors. I chose this palette ← as purple, green and pink are my favourite colours. I used rose red for the navigation bar, medium purple for the titles of the nav bar, maximum blue purple as my main background, magic mint as my aside background and apricot for the hover animations. Luckily the website converts colours into rgba for you so I didn't need to use an additional website for conversion purposes.

For my background I used unsplash to find a royalty free image for my background. Sticking to the colour theme I have, I went with wisteria for aesthetic purposes, this particular background was by user Aki Nakazawa ←

Lastly, I chose my font using google fonts . I wanted something both fluid and whimsical to fit with the theme of my website, other fonts seemed too heavy or too block like, or alternatively were too illegible from looking too much like handwriting so I chose the Shadows Into Light font as seen below, I used my favourite Shakespearean soliloquy to test out the text so I could visualise what it would look like in a full sentence/ paragraph.
